# GuinéeLogement — Plateforme Immobilière Intelligente

Bienvenue dans le projet GuinéeLogement, une plateforme moderne de location et vente immobilière en Guinée, boostée par l'Intelligence Artificielle.

## 🚀 Technologies

- **Backend** : Node.js, Express, PostgreSQL (pg native pool)
- **Frontend** : Vue.js 3, Vite, Pinia, Tailwind CSS
- **IA** : Assistant conversationnel intégré

## 📂 Structure du Projet

- `backend/` : API REST native Node.js (Authentification, Logements, Paiements, IA)
- `frontend/` : Application Web moderne et responsive

## 🛠️ Installation et Lancement

### 1. Backend
1. Entrez dans le dossier : `cd backend`
2. Installez les dépendances : `npm install`
3. Configurez votre fichier `.env` (voir `.env.example` ou le README interne)
4. Lancez le serveur : `npm run dev`

### 2. Frontend
1. Entrez dans le dossier : `cd frontend`
2. Installez les dépendances : `npm install`
3. Lancez l'application : `npm run dev`

## ✨ Fonctionnalités Clés
- **Recherche IA** : Trouvez votre logement idéal via une interface de chat.
- **Paiements Sécurisés** : Intégration de commission automatique (10%).
- **Messaging** : Discutez directement avec les propriétaires.
- **Filtres Avancés** : Recherche par ville, prix, type et critères spécifiques.
